# Data Analysis

## Data Sources
- Google Sheets: financial data, metrics
- Notion: project data, documentation
- HubSpot: CRM data, leads, deals

## Analysis Types
- Trend analysis (time series)
- Cohort analysis (user groups)
- Funnel analysis (conversion rates)
- Comparative analysis (A vs B)
