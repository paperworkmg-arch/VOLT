# React Bits — Animated Components Library

## What's Available
- 110+ customizable animations
- Text animations (typewriter, glitch, gradient, wave)
- Background effects (particles, aurora, grid, noise)
- UI components (animated cards, buttons, modals)
- Available in JS/TS with CSS or Tailwind variants

## How to Use
```bash
npm install react-bits
```

## Tools
- Background Studio: design custom backgrounds
- Shape Magic: create animated shapes
- Texture Lab: generate textures and export code

## Reference
Documentation: https://github.com/DavidHDev/react-bits
When building UI, check react-bits for existing animated components before creating custom animations.
