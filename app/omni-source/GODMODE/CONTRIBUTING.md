# Contributing to GODMODE

This is a **showcase/research project**. It reflects one developer's workflow and approach to AI-assisted development.

## Current Status

- Issues: welcome for bug reports and questions
- Pull Requests: not accepted at this time
- Feature requests: welcome as discussions

## How to Contribute

The best way to contribute right now:
1. **Try it** — install and use GODMODE in your projects
2. **Share feedback** — open an issue describing your experience
3. **Create your own skills** — build on top of GODMODE
4. **Share your setup** — tell others how you customized it

## Code of Conduct

Be respectful. This is a solo project built with passion.
